to text for formats

Converts data into simple text.

Signature

> to text {flags}

Input/output types:

input output
any string

Examples

Outputs data as simple text

>1|to text
1

Outputs external data as simple text

>githelp-a|lines|find-r'^ '|to text

Outputs records as simple text

>ls|to text
