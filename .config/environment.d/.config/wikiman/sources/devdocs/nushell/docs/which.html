which for system

Finds a program file, alias or custom command.

Signature

> which {flags} (application) ...rest

Flags

  • --all, -a: list all executables

Parameters

  • application: application
  • ...rest: additional applications

Input/output types:

input output
nothing table

Examples

Find if the 'myapp' application is available

>whichmyapp
