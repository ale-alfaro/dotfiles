hash for hash

Apply hash function.

Signature

> hash {flags}

Input/output types:

input output
nothing string

Notes

You must use one of the following subcommands. Using this command as-is will only produce this help message.

Subcommands:

name type usage
hash md5 Builtin Hash a value using the md5 hash algorithm
hash sha256 Builtin Hash a value using the sha256 hash algorithm
