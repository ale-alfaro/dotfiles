let-env for removed

`let-env FOO = ...` has been removed, use `$env.FOO = ...` instead.

Signature

> let-env {flags} (var_name) (initial_value)

Parameters

  • var_name: variable name
  • initial_value: equals sign followed by value

Input/output types:

input output
nothing nothing
