VideoStreamTheora

Inherits: VideoStream < Resource < RefCounted < Object

VideoStream resource for Ogg Theora videos.

Description

VideoStream resource handling the Ogg Theora video format with .ogv extension. The Theora codec is decoded on the CPU.

Note: While Ogg Theora videos can also have an .ogg extension, you will have to rename the extension to .ogv to use those videos within Godot.
