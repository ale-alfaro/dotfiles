9.250 SIN â€” Sine function

Description:

SIN(X) computes the sine of X.

Standard:

Fortran 77 and later

Class:

Elemental function

Syntax:

RESULT = SIN(X)

Arguments:
X The type shall be REAL or COMPLEX.
Return value:

The return value has same type and kind as X.

Example:
program test_sin
  real :: x = 0.0
  x = sin(x)
end program test_sin
Specific names:
