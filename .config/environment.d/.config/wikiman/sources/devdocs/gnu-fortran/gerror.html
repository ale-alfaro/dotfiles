9.120 GERROR â€” Get last system error message

Description:

Returns the system error message corresponding to the last system error. This resembles the functionality of strerror(3) in C.

Standard:

GNU extension

Class:

Subroutine

Syntax:

CALL GERROR(RESULT)

Arguments:
RESULT Shall of type CHARACTER and of default
Example:
PROGRAM test_gerror
  CHARACTER(len=100) :: msg
  CALL gerror(msg)
  WRITE(*,*) msg
END PROGRAM
See also:

IERRNO, PERROR
