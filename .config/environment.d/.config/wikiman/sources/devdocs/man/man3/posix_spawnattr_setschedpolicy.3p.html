NAME 

       posix_spawnattr_setschedpolicy — set the spawn-schedpolicy
       attribute of a spawn attributes object (ADVANCED REALTIME)

SYNOPSIS 

       #include <sched.h>
       #include <spawn.h>

       int posix_spawnattr_setschedpolicy(posix_spawnattr_t *attr,
           int schedpolicy);

DESCRIPTION 

       Refer to posix_spawnattr_getschedpolicy(3p).
