@DeprecatedSinceKotlin("1.5", "2.3") fun toChar(): Char
Deprecated: Direct conversion to Char is deprecated. Use toInt().toChar() or Char constructor instead.

Converts this Short value to Char.

The resulting Char code is equal to this value reinterpreted as an unsigned number, i.e. it has the same binary representation as this Short.
