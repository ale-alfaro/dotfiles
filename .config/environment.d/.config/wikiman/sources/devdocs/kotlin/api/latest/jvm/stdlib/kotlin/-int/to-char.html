Platform and version requirements: JVM (1.0), JS (1.0), Native (1.3) 
fun toChar(): Char

Converts this Int value to Char.

If this value is in the range of Char codes Char.MIN_VALUE..Char.MAX_VALUE, the resulting Char code is equal to this value.

The resulting Char code is represented by the least significant 16 bits of this Int value.
