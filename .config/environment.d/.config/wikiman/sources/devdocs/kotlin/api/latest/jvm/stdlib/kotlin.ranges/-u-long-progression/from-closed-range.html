fromClosedRange

kotlin-stdlib / kotlin.ranges / ULongProgression / fromClosedRange
Platform and version requirements: JVM (1.0), JS (1.0), Native (1.0) 
fun fromClosedRange(
    rangeStart: ULong, 
    rangeEnd: ULong, 
    step: Long
): ULongProgression

Creates ULongProgression within the specified bounds of a closed range.

The progression starts with the rangeStart value and goes toward the rangeEnd value not excluding it, with the specified step. In order to go backwards the step must be negative.

step must be greater than Long.MIN_VALUE and not equal to zero.
