Platform and version requirements: JVM (1.0), JS (1.0), Native (1.0) 
abstract fun markNow(): ComparableTimeMark

Marks a point in time on this time source.

The returned TimeMark instance encapsulates the captured time point and allows querying the duration of time interval elapsed from that point.
