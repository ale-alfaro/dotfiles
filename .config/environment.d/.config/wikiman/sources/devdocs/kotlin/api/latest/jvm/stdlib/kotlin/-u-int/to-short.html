fun toShort(): Short

Converts this UInt value to Short.

If this value is less than or equals to Short.MAX_VALUE, the resulting Short value represents the same numerical value as this UInt.

The resulting Short value is represented by the least significant 16 bits of this UInt value. Note that the resulting Short value may be negative.
