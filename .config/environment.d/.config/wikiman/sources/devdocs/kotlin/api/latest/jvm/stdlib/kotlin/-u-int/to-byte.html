Platform and version requirements: JVM (1.0), JS (1.0), Native (1.0) 
fun toByte(): Byte

Converts this UInt value to Byte.

If this value is less than or equals to Byte.MAX_VALUE, the resulting Byte value represents the same numerical value as this UInt.

The resulting Byte value is represented by the least significant 8 bits of this UInt value. Note that the resulting Byte value may be negative.
