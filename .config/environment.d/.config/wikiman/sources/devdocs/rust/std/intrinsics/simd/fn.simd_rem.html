Function simd_rem

pub unsafe fn simd_rem<T>(lhs: T, rhs: T) -> T
ðŸ”¬This is a nightly-only experimental API. (core_intrinsics)

Returns remainder of two vectors elementwise.

T must be a vector of integers or floats.

Safety

For integers, rhs must not contain any zero elements. Additionally for signed integers, <int>::MIN / -1 is undefined behavior.
