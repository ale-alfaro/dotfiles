Function atomic_xchg_seqcst

pub unsafe fn atomic_xchg_seqcst<T>(dst: *mut T, src: T) -> Twhere
    T: Copy,

Stores the value at the specified memory location, returning the old value. T must be an integer or pointer type.

The stabilized version of this intrinsic is available on the atomic types via the swap method by passing Ordering::SeqCst as the order. For example, AtomicBool::swap.
