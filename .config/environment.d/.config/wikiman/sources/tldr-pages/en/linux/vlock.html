vlock

Lock virtual console. More information: https://manned.org/vlock.

  • Lock current virtual console session:

vlock

  • Lock current virtual console session via flag:

vlock {{[-c|--current]}}

  • Lock all virtual console sessions:

vlock {{[-a|--all]}}

  • Show help:

vlock {{[-h|--help]}}

  • Show version:

vlock {{[-v|--version]}}
